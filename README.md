# Ayush & Ashish Medicine Frontend

Frontend for the medicine distributor inventory system.

## Run Locally

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

Default admin login:

```text
admin@ayushashish.com
Admin@123
```

The API base URL is configured in `config.js`.
